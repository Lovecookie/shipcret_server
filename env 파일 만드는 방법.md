## .env 파일 만드는 방법

.env 파일이 있어야 관련된 정보를 읽어서 서버를 running 시킬 수 있습니다.

```bash
경로에 .env 파일을 먼저 만듭니다.
'src/.env'

그리고 아래의 정보를 복사하여 붙여넣고
설정 값을 세팅합니다.
```

```bash
#
# DB Config
#

MYSQL_HOST=...
MYSQL_PORT=...
MYSQL_ROOT_PASSWORD=...
MYSQL_DATABASE=...
MYSQL_USER=...
MYSQL_PASSWORD=...

ACCOUNT_DB=...
CONTENT_DB=...

#
# Service Config
#

ADMIN_USER=...
ADMIN_PASSWORD=...
SERVER_PORT=...
NODE_ENV=development # 'development', 'production', 'test', 'provision'
SECRET_KEY=...
JWT_SECRET=...
```
